# Styling

Use SCSS Modules and design tokens.

## Rules

- Component styles live in `.module.scss`.
- Shared tokens live in `src/styles/globals.scss`.
- Use mobile-first responsive styles.
- Support 320px, 768px, and 1200px.
- Keep UI corporate, restrained, and scannable.
- Do not overinvest in animations, decorative polish, or product-grade design-system work beyond what the assignment requires.
- Do not add Tailwind, styled-components, or emotion.
- Do not use inline styles for ordinary styling.
